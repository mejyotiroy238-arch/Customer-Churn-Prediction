# Customer Churn Prediction for Telecom

## 📌 Project Overview

This project predicts whether a telecom customer is likely to leave (churn) using Machine Learning techniques.

The project was developed as part of an AI & Machine Learning Internship.

---

## 🎯 Objectives

- Load and clean the Telco Customer Churn dataset
- Perform Exploratory Data Analysis (EDA)
- Encode categorical variables
- Train Machine Learning models
- Optimize model performance
- Visualize feature importance

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 🤖 Machine Learning Models

- Decision Tree Classifier
- Gradient Boosting Classifier

---

## 📊 Model Evaluation

- Accuracy
- Classification Report
- Confusion Matrix
- Cross Validation
- Hyperparameter Tuning

---

## 📂 Dataset

Telco Customer Churn Dataset (Kaggle)
